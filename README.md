# Twitter Clone

Twitter clone built out with React, HTML/CSS, Material-UI/Material-Icons with a Firebase backend. 

## Usage

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

You can also view it here https://twitter-clone-45cc6.web.app/.
